# Open BMCL API for Python端

## 第一步
```
git clone https://github.com/tianxiu2b2t/python-openbmclapi.git
```
到你的项目中

先运行一次生成配置文件
```
python main.py
```
然后填写你的clusterId和clusterSceret到config.properties中

再一次运行服务器就你可以退休了（跑路）
